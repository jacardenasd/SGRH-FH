# Documentos convertidos a texto/Markdown

Estos archivos están pensados para poder copiar/pegar en chats (por ejemplo, en Visual Studio / Copilot Chat) y que el contenido sea legible y segmentado por página.

## Sugerencia de uso en Copilot/Chat
- Copia únicamente las secciones relevantes (por ejemplo “Página 3” o una cláusula).
- Si necesitas que el modelo aplique cambios, pega también tu instrucción y, si aplica, el formato objetivo (plantilla).
